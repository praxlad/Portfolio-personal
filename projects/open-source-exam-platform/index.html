Documentation & Guides

Importing & Exporting Data

Exporting: Click "Export Tests" on the Dashboard. This downloads your entire library as a `.json` file, which is useful for backing up or sharing with others.

Importing: Click "Import CSV/JSON" on the Dashboard. You can upload previously exported JSON files, or bulk-import questions via CSV.

CSV Format (Headers required):

question,option_a,option_b,option_c,option_d,correct,mediaUrl
What is 2+2?,1,2,3,4,D,
Identify the sound.,Car,Bird,Dog,Cat,B,./media/bird.mp3

Note: 'correct' must be A, B, C, or D.

Solving the Media Embedding Problem

Because this platform operates as a single offline HTML file, it relies on the browser's `localStorage` to save tests. This storage has a strict 5MB limit.

If we tried to embed heavy images or audio directly into the database (using Base64 encoding), the app would crash instantly.

The Solution: Local File Linking

  1. Create a folder on your computer (e.g. ExamPlatform).
  2. Put this HTML file inside it.
  3. Create a subfolder named media next to the HTML file.
  4. Place your files there (e.g. audio1.mp3).
  5. In the Question Builder, under "Media Link", type: ./media/audio1.mp3.

Alternatively, you can just use public URLs (e.g., https://imgur.com/...png), which requires an internet connection during the exam.

How to Use AI Integrations

Connect AI models to generate questions and analyze your performance.

1. Setup: Go to the 🔌 AI Integrations tab. Enter your API keys for Cloud models (OpenAI, Gemini, Anthropic), or use the Custom URL for free open-source endpoints like Pollinations.ai. Click 'Save Key' (these are securely stored locally on your device).

2. Performance Analysis: In Results & Analytics, select a past attempt from the left panel. Choose your preferred model, focus area, tone, and analysis length. Click 'Analyse Result' to get personalized feedback on your weak points.

3. Chat Memory: After an analysis is generated, you can use the chat box to ask follow-up questions. The AI will remember the context of your test attempt.

Upcoming / Planned Features
  • Document Parsing: Extract text directly from uploaded PDFs and Word Documents in the AI Builder, rather than needing to copy/paste text.
  • Spaced Repetition (SRS): Automatically compile incorrectly answered questions into a smart "weakness deck" to re-test you using the Leitner system.
  • Math & LaTeX Support: Integrate MathJax so engineering and science formulas render perfectly in questions and options.
  • Export to PDF: Generate a printable, beautiful scorecard and answer review sheet for your offline records.
  • Direct SQLite Support: A native desktop wrapper (via Tauri) to permanently bypass the 5MB browser storage limits, allowing Gigabytes of test data.
